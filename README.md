# Install
- `git clone https://github.com/makarasty/MindustryServerMonitoringBot` Copy bot files
- `cd ./MindustryServerMonitoringBot/` Go to bot directory
- `npm i` Install the bot components
- `>>> Configure .env <<<`
- `>>> Configure config.js <<<`
- `node .` Start the bot

# OS Dependencies
- Node.js

# Lib Dependencies
- discord.js
- dotenv
